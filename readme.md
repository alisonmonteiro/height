# height
> Apply the same height for your columns. Row by row. It's not a grid. See an example [below](https://github.com/alisonmonteiro/height#usage).


## Install

```
$ npm install --save height-js
```


## Usage

```javascript
import Height from 'height-js';

const options = {
  columns: 3,
  listElem: '.list .item'
};

Height(options);
```
