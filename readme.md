# height
> Apply the same height for your columns. Row by row.

It's not a grid. See an example [below](https://github.com/alisonmonteiro/height#usage).


## Install

```
$ npm install --save height-js
```


## Usage

```javascript
var options = {
  listElem: 'ul .item',
  columns: 3
};

Height(options);
```
